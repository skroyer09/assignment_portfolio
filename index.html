What I Learned

Throughout this theme, I learned how to best apply colours, to create visually appealing color schemes with good contrast, I also learned about the importance of accessibility in web design. by using tools like color contrast checkers to make sure that text is easily readable against its background.

I also delved deeper into CSS and learned how to use CSS grid and CSS flexbox to create responsive layouts that look great on all devices. I also learned about the best practices for creating responsive designs, such as using media queries and mobile-first design principles.

Overall, this theme taught me a lot about the importance of design in web development and how to create visually appealing and accessible websites using CSS. I feel confident in my abilities to apply these skills to my future web development projects.
